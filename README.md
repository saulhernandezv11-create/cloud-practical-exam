# Prueba técnica – Cloud Practical Exam

Solución a los 6 casos de la prueba técnica de habilidades cloud para EnvíoClick.

---

## Casos resueltos

| Caso | Descripción |
|------|-------------|
| [case-1-lambda-template](./case-1-lambda-template/) | Automatización para crear lambdas estandarizadas |
| [case-2-github-actions-lambda](./case-2-github-actions-lambda/) | Pipeline CI/CD multi-ambiente con aliases |
| [case-3-laravel-vue-cicd](./case-3-laravel-vue-cicd/) | Despliegue continuo para app Laravel + Vue |
| [case-4-fix-existing-project](./case-4-fix-existing-project/) | Corrección del pipeline cloud-lab |
| [case-5-cost-optimization](./case-5-cost-optimization/) | Análisis y propuestas de optimización de costos |
| [case-6-webhook-async](./case-6-webhook-async/) | Arquitectura asíncrona para webhook con SLA < 2s |

La documentación detallada de cada caso está en [docs/](./docs/).

## Herramientas utilizadas

- AWS (Lambda, API Gateway, SQS, DynamoDB, RDS, S3, Elastic Beanstalk, CloudWatch)
- Terraform >= 1.5
- GitHub Actions
- Python 3.13
